# Proyectos

Complete implementations demonstrating professional Java applications.

## Projects

### SistemaChat
A complete messaging system implementing:
- User management and profiles
- Message sending and receiving
- Conversation tracking
- Timestamp handling
- OOP best practices

**Files:** `Chat.java`, `Usuario.java`, `Mensaje.java`, `Main.java`

### SistemaMetro
Metro/subway system implementation:
- Lines and stations
- Route planning
- System navigation

**Files:** `App.java`, `Main.java`, `MetroSystem.java`

### RentaCar (Exam Solution)
**Complete POO exam solution** featuring:
- Vehicle hierarchy (Auto, SUV, Camioneta)
- **Strategy Pattern** - Driving modes (Eco, Normal, Sport)
- **Visitor Pattern** - Consumption calculator
- **Singleton Pattern** - System instance
- File I/O integration

This is a reference implementation of the actual course exam.

## Running the Projects

```bash
cd Proyectos/SistemaChat
javac *.java
java Main
```

## Learning Value

These projects show how individual concepts combine into real applications:
- Multiple classes working together
- Design patterns in practice
- File I/O for data persistence
- Proper encapsulation and abstraction
