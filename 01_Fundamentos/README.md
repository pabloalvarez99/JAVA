# 01 - Fundamentos

Java basics: variables, control flow, arrays, and methods.

## Exercises

| # | Topic | Difficulty | Key Concepts |
|---|-------|------------|--------------|
| 1 | Variables & Operations | ⭐ | int, double, String, operators |
| 2 | Scanner Input | ⭐⭐ | Scanner, user input, conversion |
| 3 | Conditionals | ⭐⭐ | if-else, comparison operators |
| 4 | While Loops | ⭐⭐⭐ | while, counters, validation |
| 5 | For Loops | ⭐⭐⭐ | for, iteration, patterns |
| 6 | Switch-case | ⭐⭐⭐ | switch, menus, control flow |
| 7 | Arrays | ⭐⭐⭐⭐ | array declaration, iteration, search |
| 8 | Methods | ⭐⭐⭐⭐ | parameters, return values, modular code |
| 9 | 2D Arrays | ⭐⭐⭐⭐ | matrices, nested loops |

## Compile & Run

```bash
javac Ejercicio1.java
java Ejercicio1
```

## Learning Goals

After completing these exercises, you should be able to:
- Declare and use variables of different types
- Read user input with Scanner
- Control program flow with conditionals and loops
- Work with arrays (1D and 2D)
- Create and call methods
