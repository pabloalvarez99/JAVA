# Ejercicios de POO - Nivel Medio
## Preparacion para Prueba 2 - POO

Este conjunto de ejercicios esta disenado para prepararte progresivamente para resolver examenes de Programacion Orientada a Objetos de nivel medio-avanzado.

## Estructura de los Ejercicios

Los ejercicios estan organizados en **niveles de dificultad creciente**:

### Nivel 1: Fundamentos (Ejercicios 1-3)
- Herencia basica
- Polimorfismo simple
- Uso de ArrayList

### Nivel 2: Patrones Introductorios (Ejercicios 4-6)
- Strategy Pattern
- Singleton Pattern
- Factory Pattern

### Nivel 3: File I/O y Colecciones (Ejercicios 7-8)
- Lectura de archivos estructurados
- Procesamiento de datos
- Uso de Collections Framework

### Nivel 4: Integracion de Conceptos (Ejercicios 9-10)
- Multiples patrones combinados
- Visitor Pattern
- Arquitectura de Sistema
- Problemas similares al examen

### Nivel 5: Ruteo de Codigo (Ejercicios 11-12)
- Tracing de codigo con patrones
- GUI con Swing
- Flujo de ejecucion complejo

## Conceptos Clave Cubiertos

1. **Herencia y Polimorfismo**
2. **Design Patterns:**
   - Strategy
   - Singleton
   - Factory
   - Visitor
3. **File I/O**
4. **Collections Framework**
5. **Exception Handling**
6. **GUI con Swing**
7. **Arquitectura de Software**

## Como Usar Estos Ejercicios

1. **Lee el enunciado completo** del ejercicio DOS veces
2. **Intenta resolver** por ti mismo antes de ver soluciones
3. **Compila y prueba** tu codigo
4. **Compara** con las soluciones proporcionadas
5. **Entiende el "por que"** de cada decision de diseno

## Tiempo Estimado

- Ejercicios Nivel 1-2: 30-45 minutos cada uno
- Ejercicios Nivel 3-4: 60-90 minutos cada uno
- Ejercicios Nivel 5: 45-60 minutos cada uno

**Total: ~12-15 horas de practica**

## Buenas Practicas a Seguir

- Nombrar variables y metodos de forma descriptiva (camelCase)
- Nombrar clases con PascalCase
- Usar JavaDoc para metodos publicos
- Manejar excepciones apropiadamente
- Evitar ciclos anidados cuando sea posible
- Aplicar principios SOLID
- Codigo limpio y legible

---

**Nota:** Todos los ejercicios siguen la misma estructura y filosofia del examen real de POO.
