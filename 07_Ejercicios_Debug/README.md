# 07 - Ejercicios de Debug

Learn debugging by finding and fixing bugs in broken code.

## How to Use

1. Read the exercise description
2. Try to identify the bug WITHOUT running the code
3. Run the code to confirm the bug
4. Fix the bug
5. Verify with expected output
6. Check solution in `soluciones/` folder
