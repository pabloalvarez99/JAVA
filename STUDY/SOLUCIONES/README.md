# Soluciones

Estas carpetas contienen soluciones a los ejercicios de Python y Rust.
Usalas solo para autocorreccion despues de intentar resolver.
La estructura replica las carpetas de `STUDY/PYTHON` y `STUDY/RUST`.

## Estructura
- `STUDY/SOLUCIONES/PYTHON`
- `STUDY/SOLUCIONES/RUST`

## Recomendacion
- Compara tu salida con la solucion.
- Si algo difiere, vuelve al ejercicio y corrige.
