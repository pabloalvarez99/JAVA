## Algorithm Complexities

| Algorithm | Time | Space | Type |
|-----------|------|-------|------|
| Linear Search | O(n) | O(1) | Search |
| Binary Search | O(log n) | O(1) | Search |
| BubbleSort | O(n²) | O(1) | Sort |
| MergeSort | O(n log n) | O(n) | Sort |
| QuickSort | O(n log n) | O(log n) | Sort |

- Implement common design patterns
- Handle file I/O and concurrency
