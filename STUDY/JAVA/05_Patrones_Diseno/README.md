## Exercise Progression

### Level 1: Fundamentals (Exercises 1-3)
- Inheritance basics
- Simple polymorphism
- ArrayList usage

### Level 2: Introductory Patterns (Exercises 4-6)
- **Strategy Pattern** - Behavioral flexibility
- **Singleton Pattern** - Single instance
- **Factory Pattern** - Object creation

### Level 3: File I/O & Collections (Exercises 7-8)
- Reading structured files
- Data processing
- Collections Framework

### Level 4: Integration (Exercises 9-10)
- Multiple patterns combined
- **Visitor Pattern**
- System architecture
- **Ejercicio10_RentaCarCompleto.md** = Exam replica

### Level 5: Advanced (Exercises 11-22)
- Independent practice (no solutions)
- Complex system implementations
- Code tracing with GUI

## Recommended Path

1. Read `INICIO_AQUI.md`
2. Complete exercises 1-10 (guided)
3. Review pattern guides in `guias/`
4. Practice with `preparacion_examen/` materials
5. Attempt exercises 11-22 independently
