# 04 - Funciones

Definicion:
Las funciones encapsulan logica reutilizable. Aceptan parametros, retornan
valores y pueden tener argumentos por defecto.

Ejemplos:
```python
def stats(nums):
    return min(nums), max(nums), sum(nums) / len(nums)
```
