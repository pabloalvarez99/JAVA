# 05 - Archivos y datos

Definicion:
Leer y escribir archivos permite persistir datos. CSV y JSON son formatos
comunes para intercambio.

Ejemplos:
```python
from pathlib import Path

texto = Path("datos.txt").read_text().splitlines()
print(len(texto))
```
