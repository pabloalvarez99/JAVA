# 03 - Control de flujo

Definicion:
El control de flujo define como se ejecuta el codigo: condiciones, bucles,
`break` y `continue`.

Ejemplos:
```python
for n in range(1, 6):
    if n % 2 == 0:
        continue
    print(n)
```
