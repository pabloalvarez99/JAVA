# 12 - Proyecto integrador

Definicion:
Un proyecto integrador combina conceptos de varios modulos en una solucion real.

Ejemplos:
```python
tareas = []

def agregar(titulo):
    tareas.append({"id": len(tareas) + 1, "titulo": titulo, "hecha": False})
```
