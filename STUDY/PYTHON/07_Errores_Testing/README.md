# 07 - Errores y testing

Definicion:
El manejo de errores con excepciones evita fallos silenciosos. Las pruebas
validan el comportamiento esperado.

Ejemplos:
```python
try:
    edad = int("20")
except ValueError:
    edad = 0
```
