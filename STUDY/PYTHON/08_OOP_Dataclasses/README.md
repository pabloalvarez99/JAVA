# 08 - OOP y dataclasses

Definicion:
Las clases modelan entidades con estado y comportamiento. `@dataclass`
reduce codigo repetitivo para datos.

Ejemplos:
```python
from dataclasses import dataclass

@dataclass
class Tarea:
    titulo: str
    hecha: bool = False
```
