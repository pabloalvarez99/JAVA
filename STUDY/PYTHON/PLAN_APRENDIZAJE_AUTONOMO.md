# Plan de aprendizaje autonomo: Python

Objetivo: mejorar el aprendizaje autonomo usando el plan en `STUDY/PYTHON`.
Soluciones: ver `STUDY/SOLUCIONES/PYTHON` (usar solo para autocorreccion).

## 1) Principios de trabajo
- Practica activa: escribir codigo con frecuencia y con objetivos claros.
- Error como guia: cada error debe terminar en una regla clara.
- Ejercicios primero, teoria despues: aprende con tareas concretas.
- Progreso visible: checklist y entregables por modulo.

## 2) Guia por ejercicio
1. Lee el ejercicio y escribe un plan de 3 pasos.
2. Implementa sin buscar soluciones.
3. Prueba con 2 casos extra (borde y normal).
4. Refactor minimo (nombres, funciones pequenas).
5. Resumen de 2-3 lineas en tu bitacora.

## 3) Sistema de feedback
- Bitacora de errores: error -> causa -> fix -> regla.
- Lista de "dudas abiertas": responder al final del modulo.
- Mini retro: reescribe 1 ejercicio sin mirar codigo.

## 4) Criterios de dominio por modulo (referencia `plan.md`)
- Modulo 0-1: tipos, control de flujo, listas/dicts sin ayuda.
- Modulo 2-3: funciones, archivos y CLI funcionales.
- Modulo 4: tests basicos y manejo de errores.
- Modulo 5: OOP util y dataclasses.
- Modulo 6-8: stdlib, APIs, datos y visualizacion.
- Modulo 9: proyecto integrador con README y tests.

## 5) Entregables minimos
- Un script por ejercicio clave.
- Un README corto por modulo.
- Un mini proyecto cuando el modulo lo indique.
