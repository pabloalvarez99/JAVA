# 02 - Tipos y colecciones

Definicion:
Python tiene tipos basicos (int, float, str, bool, None) y colecciones
(list, tuple, dict, set) para modelar datos.

Ejemplos:
```python
texto = "banana"
conteo = {}
for ch in texto:
    conteo[ch] = conteo.get(ch, 0) + 1
print(conteo)
```
