# 01 - Entorno y ejecucion

Definicion:
El entorno de Python es el interprete mas las dependencias del proyecto. Un venv
permite aislar paquetes y evitar conflictos.

Ejemplos:
```python
import sys

print(sys.version)
print(sys.executable)
```
