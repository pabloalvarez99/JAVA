# Plan de aprendizaje autonomo: Rust

Objetivo: mejorar el aprendizaje autonomo usando el curso en `STUDY/RUST`.
Soluciones: ver `STUDY/SOLUCIONES/RUST` (usar solo para autocorreccion).

## 1) Principios de trabajo
- Practica deliberada: objetivo claro, intento real, retroalimentacion rapida.
- Primero pensar, luego mirar: intenta resolver antes de buscar ayuda.
- Progreso visible: registra lo que aprendiste y los errores comunes.

## 2) Guia por ejercicio
1. Lee el objetivo y define 3 pasos concretos.
2. Implementa sin ver la solucion.
3. Compila y corrige hasta que pase.
4. Repite con 1 variacion (cambia datos o agrega una regla).
5. Escribe 2-3 lineas de resumen en tu bitacora.

## 3) Sistema de feedback
- Bitacora de errores: anota el error, causa, fix y regla aprendida.
- Lista de dudas: crea preguntas y responde al final del modulo.
- Mini retest: reescribe un ejercicio sin mirar el codigo anterior.

## 4) Criterios de dominio por carpeta
- 01_Fundamentos: variables, tipos, control de flujo sin errores de compilacion.
- 02_Ownership_Borrowing: puedes explicar move, borrow, lifetimes basicos.
- 03_Structs_Enums: defines structs/enums y usas Option/Result.
- 04_Pattern_Matching: match con guards y destructuring claro.
- 05_Traits_Generics: defines traits y aplicas trait bounds.
- 06_Colecciones: eliges coleccion correcta y la usas sin clones innecesarios.
- 07_Manejo_Errores: propagas errores con Result y `?`.
- 08_Iteradores_Closures: pipelines con map/filter/fold claros.
- 09_Modulos_Cargo: organizas modulos y tests en Cargo.
- 10_Concurrencia: threads, channels y locks sin deadlocks basicos.

## 5) Proyectos y cierre
- Cada carpeta termina con un mini proyecto: hazlo completo.
- Agrega un README corto: que hace, como correrlo, limites.
