# 11 - Proyectos complejos

Definicion:
Los proyectos integran conceptos en una solucion completa con entrada, datos y
manejo de errores.

Ejemplos:
```rust
struct Config { archivo: String }

fn main() {
    let cfg = Config { archivo: "datos.txt".to_string() };
    println!("{}", cfg.archivo);
}
```
