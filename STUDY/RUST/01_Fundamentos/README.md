# 01 - Fundamentos de Rust

Definicion:
Los fundamentos cubren variables, tipos, control de flujo, funciones y manejo
basico de strings y slices. Son la base para todo lo demas.

Ejemplos:
```rust
fn main() {
    let mut x = 1;
    x += 1;
    println!("x = {}", x);
}
```
