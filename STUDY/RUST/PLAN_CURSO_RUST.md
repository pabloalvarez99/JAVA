# Plan completo: Curso practico de Rust

## Vision general
- Curso organizado por conceptos en `STUDY/RUST`.
- Cada carpeta incluye README.md con definicion y ejemplo.
- Ejercicios en archivos .rs dentro de cada carpeta.
- Soluciones separadas en `STUDY/SOLUCIONES/RUST`.

## Mapa por conceptos
- 01_Fundamentos: variables, tipos, control de flujo, funciones basicas.
- 02_Ownership_Borrowing: move, borrow, lifetimes basicos.
- 03_Structs_Enums: structs, enums, Option, Result.
- 04_Pattern_Matching: match, if let, destructuring, guards.
- 05_Traits_Generics: traits, generics, trait bounds, dyn.
- 06_Colecciones: Vec, HashMap, HashSet, BTree.
- 07_Manejo_Errores: Result, ?, errores propios.
- 08_Iteradores_Closures: closures, iteradores, pipelines.
- 09_Modulos_Cargo: mod, pub, use, cargo test.
- 10_Concurrencia: threads, channels, mutex, arc, rwlock.
- 11_Proyectos_Complejos: proyectos completos con README y tests.
- 12_Ejercicios_Debug: diagnostico y correccion de bugs.

## Como usar el plan
1. Lee el README del concepto.
2. Resuelve los ejercicios en los archivos .rs de esa carpeta.
3. Usa `STUDY/SOLUCIONES/RUST` solo para autocorreccion.
