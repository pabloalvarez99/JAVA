# 12 - Ejercicios de debug

Definicion:
El debugging en Rust se enfoca en leer errores del compilador, aislar el fallo
y corregirlo con cambios pequenos.

Ejemplos:
```rust
fn main() {
    let v = vec![1, 2, 3];
    // println!("{}", v[3]); // bug: indice fuera de rango
}
```
