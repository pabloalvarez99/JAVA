# 03 - Structs y Enums

Definicion:
Structs y enums permiten definir tipos personalizados. Option y Result son enums
clave para manejar ausencia de valor y errores.

Ejemplos:
```rust
struct Punto { x: i32, y: i32 }

fn main() {
    let p = Punto { x: 1, y: 2 };
    println!("{} {}", p.x, p.y);
}
```
