# 06 - Colecciones

Definicion:
Las colecciones permiten almacenar multiples elementos: Vec, HashMap, HashSet,
BTreeMap y VecDeque.

Ejemplos:
```rust
use std::collections::HashMap;

fn main() {
    let mut m = HashMap::new();
    m.insert("a", 1);
    println!("{}", m["a"]);
}
```
